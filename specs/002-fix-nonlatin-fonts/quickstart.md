# Phase 1 Quickstart: 非英語文字表示対応

## Goal

非英語文字が豆腐にならず表示されることを、主要画面で手早く確認する。

## Prerequisites

- 作業ディレクトリ: /home/user/repository/nativox-translate

## Run

1. 開発モードで起動
   - `npm run dev`
2. 主要画面 (入力/出力/履歴) を開く
3. 以下の代表テキストを入力または貼り付ける
4. 表示されたテキストをコピーし、他の場所に貼り付けて内容が一致するか確認する

```
日本語: こんにちは、世界
中文(简体): 你好，世界
中文(繁體): 你好，世界
한국어: 안녕하세요 세계
العربية: مرحبا بالعالم
Кириллица: Привет, мир
Latin: Café déjà vu
Emoji: 😀🚀🌏
```

## Verify

- すべての文字が豆腐にならず可読である
- 入力欄、出力表示、履歴一覧のいずれでも同様に表示される
- コピー&ペースト後も文字欠落がない
