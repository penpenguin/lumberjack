Development / Run:
- npm install
- npm run dev (Electron + Vite dev server, port 5174)
- npm run build
- npm run preview

Testing / Quality:
- npm run test (Vitest jsdom)
- npm run lint
- npm run format

- ls, rg, sed, cat, git status
